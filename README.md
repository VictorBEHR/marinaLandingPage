# Marina Landing Page

Landing page estatica criada com Vite, React, Tailwind CSS e componentes Radix UI.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Deploy

Este projeto nao usa backend, banco de dados, autenticacao ou variaveis de ambiente obrigatorias.

Para publicar, gere o build com:

```bash
npm run build
```

Depois envie apenas a pasta `dist/` para o provedor de hospedagem estatica.
